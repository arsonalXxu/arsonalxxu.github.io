如何使用hugo搭建博客

如何使用Hugo搭建博客 安装Hugo install Hugo 创建一个new site hugo new site quickstart 添加一个主题 git init git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke echo 'theme = "ananke"' >> config.toml 填写博客内容 hugo new posts/my-first-post.md 编辑my-first-post.md，即填写博客内容； 启动一个Hugo服务 hugo server -D 启动成功后可以浏览器访问本地Hugo服务 自定义主题 编辑config.toml文件，theme名字为刚刚添加的主题名； languageCode = "en-us" title = "My New Hugo Site" theme = "ananke" 建立静态页面 hugo -D
